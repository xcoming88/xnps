# X-Tunnel (X-NPS & X-NPC)

X-Tunnel is a high-performance intranet penetration solution based on Node.js native `net` module. It features control channel management, dynamic port mapping, and real-time traffic analytics.

- [English Documentation](./README_EN.md)
- [中文说明文档](./README_ZH.md)

---
**Main Components:**
- **XNPS (Server)**: The central hub managing tunnels, rules, and data pairing.
- **XNPC (Client)**: The lightweight agent deployed in the internal network.
- **Web Dashboard**: Management UI for rules, logs, and real-time statistics.
